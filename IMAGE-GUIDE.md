# One Piece Portfolio - Image Setup Guide

## 📁 Folder Structure

Create an `images` folder in your project directory:

```
portfolio/
├── images/
│   ├── jolly-roger.png
│   ├── luffy.png
│   ├── zoro.png
│   ├── sanji.png
│   └── nami.png
├── index.html
├── styles.css
├── script.js
└── Gradpic Cut.jpg
```

## 🔍 Best Sources for Images

1. **PNG Repositories:**
   - pngwing.com
   - pngtree.com
   - freepngimg.com
   - cleanpng.com

2. **Search Tips:**
   - Always add "PNG transparent" to your search
   - Look for "render" or "cutout" versions
   - Choose high-quality images (at least 500x500px)
   - Prefer manga-style black and white for authentic look

3. **Alternative Search Terms:**
   - "One Piece [character name] render PNG"
   - "One Piece [character name] transparent background"
   - "One Piece [character name] cutout"

## 📝 Step-by-Step Instructions

1. **Create the images folder:**
   ```bash
   mkdir images
   ```

2. **Download each image** from the sources above

3. **Rename the files** exactly as shown:
   - jolly-roger.png
   - luffy.png
   - zoro.png
   - sanji.png
   - nami.png

4. **Place all images** in the `images` folder

5. **Refresh your browser** - the images will automatically appear!

## 🎨 Image Specifications

- **Format:** PNG (for transparency)
- **Size:** 500x500px minimum (larger is better)
- **Background:** Transparent
- **Style:** Manga or anime style (black and white preferred for manga theme)
- **Quality:** High resolution for crisp display
