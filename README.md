## 🚀 Quick Setup

### 1. Download One Piece Images

You need 5 images total:

#### Required Images:
1. **jolly-roger.png** - Straw Hat Pirates logo
2. **luffy.png** - Monkey D. Luffy's face
3. **zoro.png** - Roronoa Zoro's face
4. **sanji.png** - Sanji's face
5. **nami.png** - Nami's face

### 2. Where to Find Images

**Best Sources:**
- [PNGWing](https://www.pngwing.com/) - Search "One Piece [character] PNG"
- [FreePNGImg](https://www.freepngimg.com/) - Search "One Piece characters"
- [PNGTree](https://pngtree.com/) - Search "One Piece PNG"

**Google Image Search:**
- Search: "Straw Hat Pirates Jolly Roger PNG transparent"
- Search: "[Character Name] One Piece PNG transparent"
- Filter: Tools → Color → Transparent

### 3. Create Images Folder

```bash
mkdir images
```

### 4. Add Your Images

Place all downloaded images in the `images` folder with these exact names:
- `jolly-roger.png`
- `luffy.png`
- `zoro.png`
- `sanji.png`
- `nami.png`

### 5. Open Portfolio

Open `index.html` in your browser and enjoy! 🎉

## 🎨 Customization

### Change Your Name
Edit `index.html` line with "YOUR NAME HERE":
```html
<span class="name">YOUR NAME HERE</span>
```

### Add More Crew Members
Want to add Usopp, Chopper, Robin, etc.?
1. Download their PNG images
2. Copy a crew card in `index.html`
3. Update the image source and text

### Change Colors
The portfolio uses black & white manga theme, but you can customize:
- Edit `styles.css`
- Search for `#000` (black) and `#fff` (white)
- Replace with your preferred colors

- **Format:** PNG with transparent background
- **Size:** Minimum 500x500px (larger is better)
- **Style:** Manga or anime style preferred
- **Quality:** High resolution for crisp display

## ⚠️ Troubleshooting

### Images Not Showing?
1. Check file names match exactly (case-sensitive)
2. Verify images are in `images` folder
3. Ensure files are PNG format
4. Try hard refresh: `Ctrl + F5` (Windows) or `Cmd + Shift + R` (Mac)

### Layout Issues?
1. Clear browser cache
2. Check browser console for errors (F12)
3. Ensure all files are in correct locations

## 🌟 Tips for Best Results

✅ Use high-quality PNG images with transparent backgrounds
✅ Choose manga-style black and white images for authentic look
✅ Make sure character faces are clearly visible
✅ Keep image file sizes reasonable (under 1MB each)
✅ Test on different screen sizes
